# Sales Summary Automation

This workflow automatically monitors Google Sheets and sends a summarized report via email.

## Overview

This automation:

- Detects updates in Google Sheets
- Checks conditions using IF logic
- Counts total orders
- Sends automatic email reports using Gmail

## Workflow Structure

Google Sheets Trigger → IF Node → Data Processing → Gmail

## Purpose

This project was built to practice workflow automation using n8n and demonstrate practical automation skills.
